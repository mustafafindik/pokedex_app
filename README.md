# Pokedex App

A Pokedex Application for Demo

## Getting Started

This project is a starting point for a Flutter application.

Its a Demo, So have a just 10 Pokemon.

## Techs and Lib
- Multi language (flutter_localizations) (Just have En and TR for now)
- flutter_svg
- Landscape Support
